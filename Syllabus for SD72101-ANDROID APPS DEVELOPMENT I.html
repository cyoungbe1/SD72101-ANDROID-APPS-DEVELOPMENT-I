SD 721  Android Apps Development I

About this Course

Course Description

Android Apps Development I students will design and code mobile apps for Android phone and tablet devices, using the Android Studio development environment, which leverages XML and Java code, and APIs. Apps will include Android widgets (images, views, containers, buttons), custom icons, externalized resources, activities, intents, native databases and preferences, navigation drawers, hamburger menus, fragments, tabbed interfaces, and the processing of external web information.   (3 hours)

Cross Listing: None.

Prerequisite Courses and Skills

Prerequisite courses: SD 611.

Prerequisite skills: The ability to download and install the Android Studio software on any computer you are utilizing in this course is mandatory, as well as is a proficiency in the Java programming language.

Course Evaluation:

To facilitate this learning experience, you are required to engage in several activities each week of this term.  Generally, the components to complete each week consist of video lectures to review, homework assignments to complete using the Android Studio development environment, a graded quiz of a dozen questions, and two or three discussion threads.  These different components have been assigned specific weights that are automatically computing your grade in the course as you progress.  As you can see on the right, Homework Assignments are weighted most heavily as 55% of your final course average. This is because programming is a skill that is learned by "doing".  The Final Exam is weighted at 20% of your final grade, as it is a comprehensive programming assignment, that will require the coding of topics covered throughout the course.  Discussion threads follow since they are cumulatively weighted at 15% of your course average.  The purpose of the discussion threads is to explore the topics of the week and to engage in an insightful discussion of them with your classmates and your instructor.  Finally, weekly quizzes contribute the remaining 10% to the course average.

The intent of this course is provide you with the instruction and experience that will assist you in becoming proficient at coding Android mobile apps.  If at any time, you have any questions, please do not hesitate to share them with your instructor.  This is not an independent study course, it is a "hybrid" online course, and your instructor is here to answer your questions and to assist in making this a productive and valuable learning experience.

 

What Will I Learn?

Upon the successful completion of this course, you'll be able to formulate an Android app solution to an appropriate business problem statement, by first formulating a logical solution and then translating this solution into an Android Java application.  You will specifically be able to:

  • use an Integrated Development Environment, such as Android Studio to create, build and execute Android appss
  • design Android apps with screen layouts that are appropriate for mobile phone devices and for larger tablet devices by creating different customized layouts
  • design and code Android apps that consist of multiple screens and the logical progression through the various screens of an app
  • code efficient Android apps that utilize many different forms of input
  • code Android apps that follow good programming practices and externalize the resources used within the app
  • code Android apps that make use of native databases and other forms of persistent storage
  • code Android apps that include navigation drawer interfaces for manipulating user traversal among various features of an app
  • code Android apps that access and manipulate web server information
  • code Android apps that include tabbed activity interfaces for facilitating user movement through the components of an app

Course Format

This is a hybrid online, asynchronous class, which means that most of our interactions will take place using Canvas and other digital tools. There are two scheduled face-to-face class meetings in the term. You are free to work on course activities at your own convenience as long as you meet the participation requirements and deadlines associated with each assignment.

This format provides a great deal of freedom for students to adopt study practices that work best for them. From experience, we can say that students will, on average, devote approximately 8-10  hours per week to this course. This estimate will vary from week to week and from student to student depending on prior experience, the nature of the topic being studied, and other factors.
